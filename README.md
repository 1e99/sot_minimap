# Sot Minimap
A minimap for the pirate game Sea of Thieves.

## Installation
1. Clone the repository
```bash
git clone https://github.com/c-128/sot_minimap.git
```

2. Build the binary
```bash
zig build
```

## Usage
To run the project, use the following command:
```
./zig-out/bin/sot_map
```

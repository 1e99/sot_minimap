# Sot Minimap
A minimap for the pirate game Sea of Thieves.

## Installation
1. Clone the repository
```bash
# Clone this repository
git clone https://github.com/c-128/sot_minimap.git
cd sot_minimap

# Build the binary
zig build

# Run the app
./zig-out/bin/sot_map
```
